# Contributing to Home-Automation (HV3.0)

Thank you for considering contributing to the Home-Automation (HV3.0) firmware. By contributing code, documentation, or other materials to this repository you agree to license your contributions under the same license as the project: CC BY-NC 4.0.

Guidelines:
- Fork the repository and open a pull request (PR).
- All code contributions must be compatible with CC BY-NC 4.0 (non-commercial). If you want to contribute under different terms, contact the maintainer.
- Use `git commit --signoff` to accept the Developer Certificate of Origin or add a brief statement in your PR that you agree to the license.
- Provide a clear description of changes and testing done.

By submitting a PR, you assert that you are authorized to contribute the code and that the code is your own or used with permissive rights.

For commercial contributors: If you need a commercial license for your contributions, please open a 'Commercial license request' issue using the template in `.github/ISSUE_TEMPLATE/commercial-license-request.md`.
